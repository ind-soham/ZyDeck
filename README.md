# ZyDeck

> A premium programmable macropad engineered for creators, developers, and makers.

## Overview

ZyDeck is a custom-designed macropad built from the ground up for the Hack Club Hackpad program. The project combines custom PCB design, firmware development, and industrial design into a polished hardware product.

## Planned Features

- [ ] 6 Mechanical Keys
- [ ] Rotary Encoder
- [ ] OLED Display
- [ ] USB-C Connectivity
- [ ] RGB Underglow
- [ ] Custom PCB
- [ ] Custom 3D Printed Case
- [ ] QMK Firmware
- [ ] Vial Support
